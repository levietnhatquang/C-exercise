  3. Áp dụng Linux Kernel Coding Style cho các mã nguồn
  4. Hoàn thành mini-project “Simple Shell”.

  Ứng viên cần phát triển thêm các lệnh sau cho “Simple Shell”:
  
    - history
    
    - cat
    
    - echo
